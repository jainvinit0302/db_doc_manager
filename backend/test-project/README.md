# test-project

DBDocManager Project

## Quick Start

1. Edit your schema in `schemas/example.yaml`
2. Validate: `dbdoc validate schemas/example.yaml`
3. Generate docs: `dbdoc generate schemas/example.yaml`
4. View docs: `dbdoc serve`
